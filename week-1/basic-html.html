HTML

HTML (Hypertext Markup language) is the set of markup symbols ot codes inserted in a file intended for display on a World Wide Web browser page. The markup tells the Web browser how to display a Web page's word and images for the user. Each indivitual markup code is referred to as an element (but many people also refer to is as a tag). Some elements come in pairs that indicate when some display effect is to begin and when it is to end.

CSS

CSS is a stylesheet language that describes the presentation of an HTML (or XML) document. CSS describes how elements must be rendeted on screen, on paper, or in other media.
